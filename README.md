# 🏨 Otel Rezervasyon Sistemi (C++)

Bu proje, C++ kullanılarak geliştirilmiş basit bir otel rezervasyon sistemidir. Nesne yönelimli programlama (OOP) prensiplerini uygulamak amacıyla ödev kapsamında geliştirilmiştir.

## 🚀 Proje Amacı

Bu projede amaç:

* OOP (Object-Oriented Programming) kavramlarını uygulamak
* Dosya işlemleri (file handling) kullanmak
* Exception handling (hata yönetimi) mekanizmasını öğrenmek
* Gerçek hayata yakın bir senaryo üzerinden pratik yapmak

## 🧩 Kullanılan Teknolojiler ve Yapılar

* C++
* OOP (Inheritance, Polymorphism, Encapsulation)
* Abstract Class ve Virtual Functions
* Exception Handling
* File Operations (`fstream`)
* Templates
* Smart Pointer (`unique_ptr`)

## ⚙️ Özellikler

* Tek kişilik ve çift kişilik oda yönetimi
* Müşteri bilgisi ekleme
* TC kimlik doğrulama (format kontrolü)
* Dosyaya müşteri bilgisi kaydetme
* Oda rezervasyonu yapma
* Oda doluluk kontrolü
* Basit ödeme sistemi (kredi kartı)

## 📂 Proje Yapısı

Tüm kod tek bir dosya içinde yazılmıştır. (Geliştirme aşamasında modüler yapıya geçirilebilir.)

## ⚠️ Notlar

* Bu proje bir öğrenme projesidir ve profesyonel kullanım için optimize edilmemiştir.
* Dosya yolu şu an **lokal bilgisayara bağlıdır** (hardcoded path).
  Farklı ortamlarda çalıştırmak için güncellenmesi gerekir.

## 🔧 Geliştirmeye Açık Alanlar

Bu proje aşağıdaki yönlerden geliştirilebilir:

* 🔹 Kodun modüler hale getirilmesi (`.h` ve `.cpp` dosyalarına ayrılması)
* 🔹 Kullanıcı arayüzünün iyileştirilmesi (menü sistemi eklenmesi)
* 🔹 Veritabanı entegrasyonu (SQLite / MySQL)
* 🔹 Dosya yolu bağımlılığının kaldırılması
* 🔹 Daha kapsamlı hata kontrol mekanizmaları
* 🔹 Birden fazla rezervasyon yönetimi
* 🔹 Test senaryolarının eklenmesi
